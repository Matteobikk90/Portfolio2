Unfortunately there is not an online version for this project but.....

Follow the instructions below

  • Download or fork a copy from here:

    https://github.com/Matteobikk90/Project3/tree/Production

  • Go inside codehunt folder with the terminal untill you will find 2 folders called api and front-end

  • Press CMD + t to open a new window

  • In one go trought the api folder

  • In the other go trought the front-end folder

  • In the api write npm install first and then nodemon app.js

  • In the front-end write python -m SimpleHTTPServer 9000

  • Now open your browser and go to http://localhost:9000/ to see my Node.JS project :-)
